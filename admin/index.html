Dashboard quản trị nội dung (MongoDB)

Đăng nhập

Dashboard gọi API Render và lưu nội dung vào MongoDB.

Nội dung website

Sau khi lưu, trang vibecoffee.vn sẽ tự đọc nội dung mới từ API khi tải lại trang.

Endpoint: /v1/admin/site/content
Tip: Render env cần có ADMIN_PASSWORD, ADMIN_JWT_SECRET; CORS include https://vibecoffee.vn.
